Privacy Policy

MCP Auth Playground – Local Bridge (Chrome Extension)

Last updated: February 20, 2026

Overview

The MCP Auth Playground – Local Bridge Chrome extension is an open-source developer tool that allows the hosted MCP Auth Playground web application to communicate with services running on your local machine or Docker network.

Data Collection

This extension does not collect, store, transmit, or share any personal data or browsing information. Specifically:

  • No analytics or telemetry of any kind.
  • No cookies or local storage used by the extension itself.
  • No data is sent to any third-party server.
  • No user accounts or sign-in required.

How the Extension Works

The extension acts as a local request bridge. When the MCP Auth Playground web app (running in your browser tab) needs to reach a URL that the browser's same-origin policy would normally block (e.g., localhost), the request is forwarded to the extension's background service worker, which performs the HTTP fetch and returns the response back to the web app.

All communication happens entirely within your browser — between the web page and the extension. No data leaves your machine beyond the HTTP requests you explicitly initiate through the playground interface.

Host Permissions

The extension requests access to all URLs (<all_urls>) because users may point the playground at any MCP server — local or remote. The extension only makes requests when explicitly triggered by user action in the playground app. It never makes requests on its own.
